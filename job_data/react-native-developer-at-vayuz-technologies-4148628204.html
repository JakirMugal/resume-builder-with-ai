      Responsibilities Include

      • Designing, developing & deploying the product
      • Work as part of a team to build React Native iOS / Android applications
      • Architect, build and maintain excellent React Native applications with clean code
      • Translating wireframes and PSD designs into functional responsive apps, Implement pixel perfect UI's that match designs
      • Binding UI elements to JavaScript object models, work with native modules when required
      • Creating RESTful services with Node.js
      • Developing scalable web architectures
      • Working in a cross-functional team to deliver a complete user experience
      • Creating unit and integration tests to ensure the quality of code
      • Being responsive to change requests and feature requests
      • Writing code that is cross-platform and cross-device compatible
      • Co-ordinating end-to-end implementation
      • Release applications to iOS and Google Play stores

      Skills:- React Native and NodeJS (Node.js)
